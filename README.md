## 📌 About the Project

The **File Organizer Tool** is a Python-based automation project designed to organize files in a selected directory into structured folders based on file types such as Images, Documents, and Videos.

This project demonstrates practical usage of Python file handling and introduces GUI development using Tkinter, making it both functional and user-friendly.

### 🚀 Key Highlights

* Automates file organization process
* Reduces manual effort and saves time
* Uses Python built-in modules (`os`, `shutil`)
* Includes a simple graphical user interface
* Beginner-friendly and practical project

### 🎯 Purpose

The main goal of this project is to solve the common problem of unorganized folders and to apply Python concepts in a real-world scenario.
